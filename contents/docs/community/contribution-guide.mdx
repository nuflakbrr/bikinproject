Welcome to the `Contribution Guide` section. We appreciate your interest in contributing to our project. This guide will help you get started with the contribution process and provide you with all the information you need to make a meaningful contribution.

Our project is open-source, and we welcome contributions from the community. Whether you're a developer, designer, or writer, there are many ways you can contribute to our project. This guide will help you understand the contribution process and provide you with the necessary information to get started.

## How to Contribute

Contributing to our project is easy. Here are some ways you can get involved:

- **Report Bugs**: If you encounter a bug or issue with our project, please report it on our [GitHub repository](https://github.com/nufalkbrr/bikinproject/issues).
- **Fix Bugs**: If you're a developer, you can help us fix bugs by submitting patches or pull requests on our [GitHub repository](https://github.com/nufalkbrr/bikinproject/issues).
- **Request Features**: If you have an idea for a new feature or improvement, please let us know by creating a feature request on our [GitHub repository](https://github.com/nufalkbrr/bikinproject/issues).
- **Submit Pull Requests**: If you'd like to contribute code to our project, you can submit a pull request on our [GitHub repository](https://github.com/nufalkbrr/bikinproject/pulls).
- **[Create New Templates](#added-new-templates)**: You can create new templates for our project and submit them for inclusion in our project.
- **Write Documentation**: If you're a writer or technical writer, you can help improve our documentation by submitting updates or new content.
- **Improve Documentation**: If you're a writer or technical writer, you can help us improve our documentation by submitting updates or new content.

## Added New Templates

### Next.js

1. Clone the repository.

```bash
  git clone https://github.com/nuflakbrr/next-template.git
```

2. Create new branch.

```bash
  git checkout -b template_name-version
```

3. Create new template.

4. Push to the branch.

```bash
  git push origin template_name-version
```
